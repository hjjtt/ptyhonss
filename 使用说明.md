## 1. 运行环境要求

| 项目 | 要求 |
|---|---|
| 操作系统 | Windows / macOS / Linux 均可 |
| Python 版本 | **3.10 及以上** |
| 网络 | 需能访问 `api.jikan.moe`（首次抓取时） |
| 磁盘空间 | 约 50 MB（数据库 + 原始缓存 + 图表） |
| 额外服务 | **无需**（不依赖 MySQL/Redis 等任何服务端） |

## 2. 安装步骤

### 第 1 步：确认 Python 版本

```bash
python --version
# 应输出: Python 3.10.x 或更高
```

### 第 2 步：安装依赖（仅 5 个包）

在项目根目录执行：

```bash
pip install -r requirements.txt
```

依赖清单：

| 包 | 用途 |
|---|---|
| `requests` | HTTP 请求 |
| `pandas` | 数据处理、Excel 读写 |
| `numpy` | 数值计算 |
| `matplotlib` | 生成分析图 |
| `openpyxl` | Excel 文件写入引擎 |

> `sqlite3` 是 Python 标准库自带的，无需单独安装。

### 第 3 步：验证安装

```bash
python -c "import requests, pandas, numpy, matplotlib, openpyxl; print('依赖安装成功')"
```

输出 `依赖安装成功` 即表示环境就绪。

## 3. 快速开始（3 分钟跑起来）

在项目根目录执行：

```bash
python main.py --skip-crawl
```

这条命令会**跳过网络抓取**，直接用项目自带的本地数据（`data/raw/` 中的缓存）完成：

1. 初始化数据库
2. 加载已有评论
3. 导出 `output/data.xlsx`
4. 生成 `output/charts/` 下的 7 张分析图

约 20-30 秒完成，看到 `全部完成, 耗时 XX 秒` 即成功。

> 这是**最常用的命令**，适合反复复跑、查看分析结果。

## 4. 运行模式详解

### 模式 A：完整流程（抓取 + 分析）

```bash
python main.py
```

会执行真正的网络抓取：
- 调用 Jikan API 抓取动漫列表与评论
- **默认只抓近两年数据**，每个动漫最多 1000 条（课程规则）
- 抓取是增量累积的：每次运行只抓没抓过的新动漫
- 抓完后自动入库、导出 Excel、生成图表

> ⚠️ 首次完整抓取因限速（1.5 秒/请求）需较长时间，请耐心等待。

### 模式 B：跳过抓取（推荐复跑）

```bash
python main.py --skip-crawl
```

直接用本地缓存数据进行分析，不访问网络。速度快、结果稳定，适合：
- 查看分析图表
- 修改分析逻辑后重跑
- 演示/答辩前确认成果物

### 参数总览

| 参数 | 作用 |
|---|---|
| （无参数） | 完整流程：抓取 + 入库 + 导出 + 分析 |
| `--skip-crawl` | 跳过抓取，只用本地数据 |

## 5. 配置项说明

所有可调参数集中在 **`scraper/config.py`**，修改后立即生效，无需改其他文件。

### 数据过滤规则（课程要求，默认开启）

```python
ENABLE_FILTER = True                # 总开关
MAX_REVIEWS_PER_ANIME_LIMIT = 1000  # 单动漫评论数上限
RECENT_YEARS = 2                    # 只保留近 N 年的评论
```

- 想抓全量历史数据（关闭规则）：把 `ENABLE_FILTER` 改为 `False`
- 想放宽时间窗口：把 `RECENT_YEARS` 改为 `3` 或更大

### 抓取参数

```python
MAX_ANIME = 80                    # 每轮最多新增动漫数
MAX_REVIEWS_PER_ANIME = 100       # 单动漫 API 翻页上限
REQUEST_INTERVAL = 1.5            # 请求间隔（秒）
MAX_RETRIES = 5                   # 失败重试次数
```

### 评分归一化（1-10 → 1-5）

```python
RATING_RANGES = [
    (9, 5),  # MAL ≥9 分 → 5 星
    (7, 4),  # MAL ≥7 分 → 4 星
    (5, 3),  # MAL ≥5 分 → 3 星
    (3, 2),  # MAL ≥3 分 → 2 星
    (0, 1),  # 其他 → 1 星
]
```

| `output/charts/02_daily_trend.png` | 单动漫每日评论趋势 + 7 日滑动平均 |
| `output/charts/03_rating_distribution.png` | 评分分布直方图 |
| `output/charts/04_monthly_trend.png` | 评论数量月度趋势 |
| `output/charts/05_type_comparison.png` | 动漫类型对比（双 Y 轴） |
| `output/charts/06_rating_vs_length.png` | 评分 vs 字数 + Reactions 散点 |

|---|---|

## 7. 常见问题（FAQ）

### Q1：运行报错 `ModuleNotFoundError`

**原因**：依赖未安装。

**解决**：

```bash
pip install -r requirements.txt
```

### Q2：抓取时一直卡住或返回 429

**原因**：触发 Jikan 限速（3 req/s）。

**解决**：程序已内置 1.5 秒/请求的节流，正常不会触发。若仍出现，稍等几分钟后重试即可。程序会自动重试，无需干预。

### Q3：完整抓取要多久？

受 `REQUEST_INTERVAL = 1.5` 秒/请求限制：
- 抓 1 个动漫 ≈ 6 个请求（1 列表 + 5 评论页）≈ 10 秒
- 80 个动漫 ≈ 13-15 分钟

### Q4：想清空数据从头抓怎么办？

删除以下文件后重跑 `python main.py`：

```
data/comments.db
data/raw\*.json
```

或者把 `scraper/config.py` 的 `ENABLE_FILTER` 改为 `False` 抓全量历史数据做对比。

### Q5：数据库怎么查看？

推荐用免费的 **DB Browser for SQLite**（https://sqlitebrowser.org）：
1. 打开软件
2. File → Open Database → 选择 `data/comments.db`
3. 在 "Browse Data" 下拉框选 `v_comment_full` 即可看到完整评论数据

### Q6：图表中文显示成方框

**原因**：系统缺少中文字体。

**解决**：代码已自动检测并使用微软雅黑/黑体等中文字体。若仍异常，请确保系统安装了 "Microsoft YaHei" 或 "SimHei" 字体。

### Q7：关闭过滤规则抓全量数据

编辑 `scraper/config.py`：

```python
ENABLE_FILTER = False   # 改成 False
```

然后 `python main.py` 即可抓取全量历史评论（不受 1000 条/近两年限制）。

### Q8：如何只重新生成图表？

```bash
python main.py --skip-crawl
```

它会用现有数据库重新出图，不重新抓取。

## 8. 目录结构与功能对应

```
ptyhonss/
├── main.py                       ← 入口, 运行这个文件
├── requirements.txt              ← 依赖清单, pip install 用
│
├── scraper/                      ← 爬虫层
│   ├── config.py                ← ★ 所有配置都在这改
│   ├── http_client.py           ← HTTP 客户端 (限速/重试)
│   ├── crawler.py               ← 抓取逻辑 + 数据过滤
│   └── cache.py                 ← 本地缓存加载
│
├── database/                     ← 存储层
│   ├── schema.sql               ← 表结构 (无需手动改)
│   ├── db_manager.py            ← 数据库读写
│   └── exporter.py              ← 导出 Excel
│
├── analysis/visualizer.py        ← 分析出图 (7 张)
│
├── data/                         ← 数据目录 (自动生成)
│   ├── comments.db              ← SQLite 数据库
│   └── raw/                     ← 原始 JSON 缓存
│
├── output/                       ← 成果物 (自动生成)
│   ├── data.xlsx / data.xlxs    ← Excel 数据文件
│   └── charts/                  ← 7 张分析图
│
└── logs/run.log                  ← 运行日志
```
